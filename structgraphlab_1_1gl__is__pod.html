template<typename T>
struct graphlab::gl_is_pod< T >

Tests if T is a POD type.

gl_is_pod<T>::value is true if T is a POD type (as determined by boost::is_pod) or if T inherits from IS_POD_TYPE. gl_is_pod<T>::value is false otherwise.
